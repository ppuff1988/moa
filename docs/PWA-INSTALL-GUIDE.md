# PWA 快速安裝指南

## 📱 如何將 MOA 安裝到您的裝置

### iPhone / iPad (iOS)

1. 使用 **Safari** 瀏覽器打開 [https://moa.sportify.tw](https://moa.sportify.tw)
2. 點擊底部工具列的 **「分享」按鈕** 📤（中間的向上箭頭）
3. 向下滾動選單，找到並點擊 **「加入主畫面」**
4. 可以自訂名稱，然後點擊 **「加入」**
5. ✅ 完成！MOA 圖示現在會出現在您的主畫面

**提示**：從主畫面打開時，MOA 會以全螢幕模式運行，就像原生 App 一樣！

---

### Android 手機 (Chrome)

1. 使用 **Chrome** 瀏覽器打開 [https://moa.sportify.tw](https://moa.sportify.tw)
2. 點擊右上角的 **選單按鈕** ⋮（三個點）
3. 選擇 **「安裝應用程式」** 或 **「新增至主畫面」**
4. 在彈出的對話框中點擊 **「安裝」**
5. ✅ 完成！MOA 圖示現在會出現在您的主畫面和應用程式列表中

**提示**：某些 Android 設備可能會在您訪問網站時自動顯示「安裝」提示橫幅。

---

### Windows 電腦 (Chrome / Edge)

1. 使用 **Chrome** 或 **Edge** 瀏覽器打開 [https://moa.sportify.tw](https://moa.sportify.tw)
2. 在網址列右側會出現 **安裝圖示** ➕
3. 點擊該圖示
4. 在彈出的對話框中點擊 **「安裝」**
5. ✅ 完成！MOA 會作為獨立應用程式出現在您的開始選單和桌面

**提示**：安裝後可以像其他 Windows 應用程式一樣固定到工作列或設定開機自動啟動。

---

### Mac 電腦 (Chrome / Edge)

1. 使用 **Chrome** 或 **Edge** 瀏覽器打開 [https://moa.sportify.tw](https://moa.sportify.tw)
2. 在網址列右側會出現 **安裝圖示** ➕
3. 點擊該圖示
4. 在彈出的對話框中點擊 **「安裝」**
5. ✅ 完成！MOA 會作為獨立應用程式出現在您的應用程式資料夾和 Dock

**提示**：可以將 MOA 固定到 Dock 以便快速訪問。

## ❓ 常見問題

### Q: 為什麼看不到安裝選項？

A: 請確保：

- 使用支援的瀏覽器（Chrome、Edge、Safari）
- 網站使用 HTTPS 連線
- 尚未安裝過 MOA（已安裝的話安裝按鈕不會顯示）

### Q: 安裝後如何更新？

A: MOA 會自動在背景檢查更新。當有新版本時，會在應用程式右下角顯示更新提示。

### Q: 如何解除安裝？

- **iOS**: 長按圖示 → 選擇「移除 App」
- **Android**: 長按圖示 → 拖曳到「解除安裝」
- **Windows**: 設定 → 應用程式 → 找到 MOA → 解除安裝
- **Mac**: 從應用程式資料夾移除（或在瀏覽器的應用程式管理中移除）

### Q: 安裝後可以離線使用嗎？

A: 部分功能可以離線使用。遊戲本身需要網路連線，但應用程式的基本界面和資源會被快取，即使在網路不穩定時也能快速載入。

### Q: 安裝會佔用多少空間？

A: 非常少！PWA 主要快取網頁資源，通常不到 10MB。
